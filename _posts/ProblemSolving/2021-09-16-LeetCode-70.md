# 문제
- https://leetcode.com/problems/climbing-stairs/
- 1,2 스텝으로 계단을 오를 수 있는 경우의 수 찾기

# 접근
- 동적 계획법을 이용하여 이전의 값을 활용해 정답을 도출

```java
class Solution {
    public int climbStairs(int n) {
        int answer[] = new int[46];
        answer[1] = 1;
        answer[2] = 2;
        
        for(int i=3; i<=45;i++)
            answer[i] = answer[i-2] + answer[i-1];
        
        return answer[n];
        
    }
}
```
