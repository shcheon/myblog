# 문제
- https://leetcode.com/problems/add-binary/
- 2의 지수승의 숫자인지 판별하는 문제

# 접근
- bit mask 연산을 통해 2의 지수승인지 체크.


```java
class Solution {
    public String addBinary(String a, String b) {
        int i,j;
        
        i = a.length() - 1;
        j = b.length() - 1;
        
        
        int carry = 0, res = 0;
        StringBuilder answer = new StringBuilder();
        while(i >= 0 || j >= 0)
        {
            int operA = 0, operB = 0;
            if(i>=0)
                operA = a.charAt(i) - '0';
            
            if(j>=0)
                operB = b.charAt(j) - '0';
            
            res = operA + operB + carry;
            
            if(res > 1){
                carry=1;
                res = res % 2;
            } else
                carry = 0;
                        
            if(i >= 0)
                i--;
            if(j >= 0)
                j--;
            
            answer.append(res);
        }
        if(carry == 1)
            answer.append("1");
        
        return answer.reverse().toString();
    }
}
```
